### 2.2 执行成功后的处理步骤

当子任务执行成功（`status = 'completed'`）时，按以下步骤处理：

#### 步骤 1：保存执行结果到 `agentSubTasks.executionResult`
```typescript
// executionResult 存储 JSON 字符串，格式：
{
  "stepOutput": "步骤执行结果文本",
  "articleMetadata": { /* article_metadata 对象 */ },
  "completedAt": "2026-02-27T10:30:00.000Z",
  "status": "success"
}
```

#### 步骤 2：更新 `dailyTasks.completedSubTasks`
- 查询当前子任务的 `orderIndex`
- 更新 `dailyTasks.completedSubTasks = orderIndex`

#### 步骤 3：更新 `dailyTasks.completedSubTasksDescription`
- 使用子任务的 `taskTitle` 或 `taskDescription`
- 格式：`"步骤 {orderIndex}: {taskTitle}"`

#### 修改 2：新增 `updateDailyTaskProgress()` 函数

**新增函数**：
```typescript
/**
 * 更新 dailyTasks 表的进度信息
 */
private async updateDailyTaskProgress(task: typeof agentSubTasks.$inferSelect) {
  console.log(`[SubtaskEngine] 更新 dailyTasks 进度: orderIndex = ${task.orderIndex}`);

  try {
    // 1. 查询当前子任务的 taskTitle
    const subTask = await db
      .select()
      .from(agentSubTasks)
      .where(eq(agentSubTasks.id, task.id))
      .limit(1);

    if (subTask.length === 0) {
      console.warn(`[SubtaskEngine] 未找到子任务: ${task.id}`);
      return;
    }

    const currentTask = subTask[0];
    const completedSubTasksDescription = `步骤 ${currentTask.orderIndex}: ${currentTask.taskTitle}`;

    // 2. 更新 dailyTasks 表
    await db
      .update(dailyTasks)
      .set({
        completedSubTasks: currentTask.orderIndex,
        completedSubTasksDescription: completedSubTasksDescription,
        updatedAt: new Date(),
      })
      .where(eq(dailyTasks.id, currentTask.commandResultId));

    console.log(`[SubtaskEngine] dailyTasks 进度已更新: ${completedSubTasksDescription}`);
  } catch (error) {
    console.error(`[SubtaskEngine] 更新 dailyTasks 进度失败:`, error);
  }
}
```

**注意**：需要引入 `dailyTasks` 表：
