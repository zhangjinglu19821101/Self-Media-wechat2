### responseStatus 判断规则：
- **有 mcpParams，且 capability_list 中有对应的 MCP 功能** → responseStatus = "pre_completed"（MCP 会执行）

- responseStatus 与 isCompleted 无关
- MCP 在后续的执行结果，不在你考虑的范围

### isCompleted 的含义（仅用于说明）
- 仅用于说明任务是否已完成（但任务是否完成不影响 responseStatus）
