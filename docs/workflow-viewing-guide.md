# 📊 工作流程执行过程查看指南

## 🎯 快速开始

### 方式1：从主页进入

1. 访问系统主页：http://localhost:5000
2. 点击顶部导航栏的 **"工作流程"** 按钮
3. 进入工作流程管理页面

### 方式2：直接访问

- 工作流程列表：http://localhost:5000/workflow
- 工作流程详情：http://localhost:5000/workflow/[工作流程ID]

---

## 📋 查看执行过程的步骤

### 第一步：查看工作流程列表

在工作流程列表页面，你会看到：
- 📊 统计卡片（总数、进行中、已完成、成功率）
- 📝 工作流程列表（标题、状态、进度、发起者）
- 🔘 操作按钮（查看执行、预览）

### 第二步：点击"查看执行"

在工作流程列表中，点击任意工作流程的 **"查看执行"** 按钮，进入详情页面。

### 第三步：实时监控执行进度

在详情页面，你可以看到：

#### 1️⃣ 状态卡片
- **当前状态**：进行中/已完成/失败/已暂停
- **执行进度**：当前步骤数 / 总步骤数（带进度条）
- **开始时间**：工作流程启动时间
- **发起者**：发起该工作流程的 Agent

#### 2️⃣ 执行流程图
10 个步骤的实时执行状态：
- ✅ **绿色**：已完成
- 🔵 **蓝色**：进行中（带动画脉冲）
- 🔴 **红色**：失败
- ⚪ **灰色**：待启动

#### 3️⃣ 每个步骤的详细信息
- **步骤名称**：该步骤的名称
- **步骤描述**：该步骤的具体任务
- **状态标签**：进行中/已完成/失败/待启动
- **Agent 流转**：发起者 → 执行者
- **预计时长**：该步骤的预估时间
- **执行结果**：如果已完成，显示执行结果
- **反馈信息**：如果有反馈，显示反馈内容
- **时间信息**：开始时间和完成时间

#### 4️⃣ 自动刷新功能
- 点击 **"自动刷新"** 按钮，每 5 秒自动更新状态
- 点击 **"手动刷新"** 按钮，立即刷新状态
- 点击 **"暂停"** 按钮，停止自动刷新

#### 5️⃣ 操作按钮（当前步骤）
- **完成步骤**：标记当前步骤为完成，需要输入执行结果
- **标记失败**：标记当前步骤为失败，需要输入失败原因

## 💡 使用技巧

### 1. 设置自动刷新
- 打开自动刷新，无需手动点击即可实时查看进度
- 建议在步骤执行期间开启自动刷新

### 2. 手动完成步骤
- 在测试或演示时，可以手动完成步骤
- 点击 **"完成步骤"** 按钮，输入执行结果
- 系统会自动启动下一步

### 3. 查看详细结果
- 点击每个步骤展开，查看执行结果和反馈
- 了解每个步骤的详细执行情况

### 4. 返回列表
- 点击左上角的 **"返回"** 按钮，返回工作流程列表
- 查看其他工作流程的执行状态

## 🎯 10 步闭环流程说明

| 步骤 | 名称 | Agent | 描述 |
|-----|------|-------|------|
| 1 | 战略制定与下达 | A → B/C/D | A 制定战略计划，下达可执行任务 |
| 2 | 任务执行 | A → C/D | C/D 执行任务，记录数据 |
| 3 | 反馈报告 | C → A/B | C/D 提交反馈报告 |
| 4 | 经验提取 | B | B 提取经验，转化为规则 |
| 5 | 迭代方案 | B → A | B 提交迭代方案 |
| 6 | 审核决策 | A → B | A 审核方案，下达指令 |
| 7 | 规则落地 | B | B 落地规则到中台/插件 |
| 8 | 效果验证 | B → C/D | C/D 验证新规则效果 |
| 9 | 验收报告 | B → A | B 提交验收报告 |
| 10 | 结果同步 | A → 全体 | A 验收结果，同步全体系 |

## 🚀 快速测试

### 触发一个新的工作流程

1. 访问 Agent A 对话页面：http://localhost:5000/agents/A
2. 输入触发指令：`触发工作流程：测试执行 - 测试10步闭环流程`
3. 或者点击快捷按钮：🚀 触发工作流程
4. 系统会自动创建并启动工作流程
5. 点击返回或直接访问：http://localhost:5000/workflow
6. 找到新创建的工作流程，点击 **"查看执行"**

### 手动完成步骤

在工作流程详情页面：
1. 找到当前进行的步骤（蓝色边框）
2. 点击 **"完成步骤"** 按钮
3. 输入执行结果，例如："已完成战略制定，已向 B/C/D 下达任务"
4. 点击确定
5. 系统会自动启动下一步
6. 重复此过程，完成所有 10 步

## ❓ 常见问题

### Q: 为什么看不到执行过程？
A: 确保工作流程已触发。在工作流程列表中查看是否有正在运行的工作流程。

### Q: 如何让系统自动执行步骤？
A: 当前版本需要手动完成步骤。自动执行功能正在开发中。

### Q: 可以同时查看多个工作流程吗？
A: 可以。打开多个标签页，每个标签页查看一个工作流程的详情。

### Q: 如何查看历史工作流程？
A: 在工作流程列表页面，所有工作流程都会显示，包括已完成的工作流程。
