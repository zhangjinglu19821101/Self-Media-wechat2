## 🔧 API 接口

### 1. 获取所有 Agent
```http
GET /api/admin/agent-builder/agents
```

**响应**:
```json
{
  "success": true,
  "data": [
    {
      "id": "A",
      "name": "核心协调者",
      "role": "Coordinator",
      "description": "负责协调多个 Agent 的工作",
      "systemPrompt": "你是一个核心协调者 Agent...",
      "maxConcurrentTasks": 5,
      "canSendTo": ["B", "C", "D"],
      "canReceiveFrom": ["B", "C", "D"]
    }
  ]
}
```

---

### 2. 获取指定 Agent
```http
GET /api/admin/agent-builder/agent/{id}
```

**响应**:
```json
{
  "success": true,
  "data": {
    "id": "A",
    "name": "核心协调者",
    "systemPrompt": "...",
    ...
  }
}
```

---

### 3. 更新 Agent 信息
```http
PUT /api/admin/agent-builder/agent/{id}
Content-Type: application/json

{
  "name": "核心协调者",
  "systemPrompt": "更新后的提示词...",
  "maxConcurrentTasks": 5
}
```

---

### 4. 获取 Agent 能力
```http
GET /api/admin/agent-builder/agent/{id}/capabilities
```

**响应**:
```json
{
  "success": true,
  "data": {
    "base": [
      {
        "id": "task-decomposition",
        "name": "任务分解",
        "level": 90,
        "description": "将复杂需求分解为具体任务",
        "type": "base",
        "replicable": true
      }
    ],
    "domain": {
      "电商": [
        {
          "id": "ecommerce-business-rules",
          "name": "电商业务规则",
          "price": 5000,
          "domain": "电商",
          "provider": "expert",
          "version": "1.0.0"
        }
      ]
    }
  }
}
```

---

### 5. 导出基础能力
```http
GET /api/admin/capabilities/export/base?agentId=A&format=json
```

---

### 6. 导出领域能力
```http
GET /api/admin/capabilities/export/domain?domain=电商&agentId=B&format=json
```

---

### 7. 导出所有能力
```http
GET /api/admin/capabilities/export/all?format=json
```

## 🎯 主要功能

### 1. 提示词管理 ✅

**通过界面操作**:
1. 访问 `/admin/agent-builder/agent/{id}`
2. 点击"编辑"按钮
3. 修改系统提示词
4. 点击"保存"按钮

**通过 API 操作**:
```bash
curl -X PUT http://localhost:5000/api/admin/agent-builder/agent/A \
  -H "Content-Type: application/json" \
  -d '{
    "systemPrompt": "你是一个核心协调者 Agent，负责协调多个 Agent 的工作..."
  }'
```

---

### 2. 能力查看 ✅

**基础能力**:
- 访问 `/admin/agent-builder/capabilities`
- 切换到"基础能力"标签
- 查看所有 Agent 的基础能力

**领域能力**:
- 访问 `/admin/agent-builder/capabilities`
- 切换到"领域能力"标签
- 查看所有 Agent 的领域能力

---

### 3. 能力导出 ✅

**导出基础能力**:
1. 访问 `/admin/agent-builder/export`
2. 切换到"基础能力"标签
3. 选择 Agent 筛选和格式
4. 点击"导出基础能力"按钮

**导出领域能力**:
1. 访问 `/admin/agent-builder/export`
2. 切换到"领域能力"标签
3. 选择领域筛选和格式
4. 点击"导出领域能力"按钮

**导出完整能力包**:
1. 访问 `/admin/agent-builder/export`
2. 切换到"完整能力包"标签
3. 选择格式
4. 点击"导出完整能力包"按钮

## 📝 快速开始

### 1. 访问管理后台

打开浏览器访问：
```
http://localhost:5000/admin/agent-builder
```

或者在主页面点击"管理后台"按钮。

---

### 2. 查看 Agent 列表

进入管理后台后，默认显示所有 Agent 的列表，包括：
- Agent ID
- 名称
- 角色
- 描述
- 能力统计（基础能力数、领域能力数）

---

### 3. 编辑 Agent 提示词

1. 在 Agent 列表中，点击某个 Agent 的"查看详情"
2. 进入 Agent 详情页
3. 点击"编辑"按钮
4. 修改系统提示词
5. 点击"保存"按钮

---

### 4. 查看能力

**方法一：通过 Agent 详情页**
1. 进入 Agent 详情页
2. 切换到"能力"标签
3. 查看基础能力和领域能力

**方法二：通过能力管理页**
1. 访问 `/admin/agent-builder/capabilities`
2. 切换到"基础能力"或"领域能力"标签
3. 使用筛选器过滤数据

---

### 5. 导出能力

1. 访问 `/admin/agent-builder/export`
2. 选择要导出的类型（基础/领域/完整）
3. 配置导出选项
4. 点击导出按钮
5. 文件会自动下载
