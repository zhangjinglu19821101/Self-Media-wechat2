# 指令格式支持更新

## 问题描述

用户反馈：某些指令格式无法触发弹框，特别是使用中文序号的格式。

## 原因分析

原正则表达式只支持以下格式：
- `# Agent C`
- `## Agent C`

但用户使用的格式是：
- `## 一、AI事业部Agent C（内容运营）专属指令`
- `## 二、AI事业部Agent D（内容生成）专属指令`
- `## 三、保险事业部Agent insurance-c（内容运营）专属指令`
- `## 四、保险事业部Agent insurance-d（内容生成）专属指令`
- `## 五、架构师B（技术支撑）专属指令`

## 解决方案

更新了所有 Agent 的正则表达式，支持以下格式：

### 1. 原有格式（继续支持）
- `# Agent C`
- `## Agent C`

### 2. 新增格式（中文序号）
- `## 一、AI事业部Agent C`
- `# 一、AI事业部Agent C`
- `## 二、架构师B`
- `# 二、架构师B`

## 更新的 Agent

### Agent C（AI运营Agent）
```typescript
// 新增正则
/##\s*[一二三四五六七八九十]+、\s*(?:AI事业部\s*)?Agent\s+C[^#\n]*\n([\s\S]*)/m,
/#\s*[一二三四五六七八九十]+、\s*(?:AI事业部\s*)?Agent\s+C[^#\n]*\n([\s\S]*)/m,
```

### Agent D（AI内容生成Agent）
```typescript
// 新增正则
/##\s*[一二三四五六七八九十]+、\s*(?:AI事业部\s*)?Agent\s+D[^#\n]*\n([\s\S]*)/m,
/#\s*[一二三四五六七八九十]+、\s*(?:AI事业部\s*)?Agent\s+D[^#\n]*\n([\s\S]*)/m,
```

### Agent insurance-c（保险运营Agent）
```typescript
// 新增正则
/##\s*[一二三四五六七八九十]+、\s*(?:保险事业部\s*)?Agent\s+insurance-c[^#\n]*\n([\s\S]*)/m,
/#\s*[一二三四五六七八九十]+、\s*(?:保险事业部\s*)?Agent\s+insurance-c[^#\n]*\n([\s\S]*)/m,
```

### Agent insurance-d（保险内容创作Agent）
```typescript
// 新增正则
/##\s*[一二三四五六七八九十]+、\s*(?:保险事业部\s*)?Agent\s+insurance-d[^#\n]*\n([\s\S]*)/m,
/#\s*[一二三四五六七八九十]+、\s*(?:保险事业部\s*)?Agent\s+insurance-d[^#\n]*\n([\s\S]*)/m,
/##\s*[一二三四五六七八九十]+、\s*保险内容创作Agent[^#\n]*\n([\s\S]*)/m,
```

### Agent B（架构师B）
```typescript
// 新增正则
/##\s*[一二三四五六七八九十]+、\s*(?:架构师B|Agent\s+B|技术负责人)[^#\n]*\n([\s\S]*)/m,
/#\s*[一二三四五六七八九十]+、\s*(?:架构师B|Agent\s+B|技术负责人)[^#\n]*\n([\s\S]*)/m,
```

## 支持的中文序号

- 一、二、三、四、五、六、七、八、九、十

## 清理逻辑更新

同时更新了 `cleanCommandContent` 函数，使其能够识别中文序号格式的 Agent 标题，并正确截取指令内容。

## 使用示例

### 格式 1：原有格式（支持）
```markdown
# Agent C
任务内容...
```

### 格式 2：二级标题（支持）
```markdown
## Agent C
任务内容...
```

### 格式 3：中文序号（新增）
```markdown
## 一、AI事业部Agent C（内容运营）专属指令
任务内容...
```

### 格式 4：单井号中文序号（新增）
```markdown
# 一、架构师B（技术支撑）专属指令
任务内容...
```

## 测试

刷新页面，重新输入指令，应该能够正常触发弹框了。

## 兼容性

✅ 向后兼容：原有格式继续支持
✅ 新增支持：中文序号格式
✅ 灵活性：支持单井号和双井号

## 注意事项

1. 中文序号必须使用顿号（、）分隔
2. 序号后可以有空格
3. Agent 名称必须包含在正则表达式中定义的关键词
4. 标题必须以换行符结束
