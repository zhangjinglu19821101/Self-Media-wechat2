# 降级逻辑 - 修正版（更安全）

## ⚠️ 发现问题

**用户的好问题：**
> "场景 1 Agent B API 报错（网络问题） pre_completed → completed —— agent B 出错了，怎么还能更新为 completed ？"

**确实不合理！** ❌

---

## 🎯 修正后的降级逻辑

### 核心原则

**当 Agent B 评审失败时，我们应该谨慎处理，而不是草率决策！**

### 修正后的降级行为

| 状态 | 新逻辑错误 | 降级行为（修正后） | 说明 |
|------|-----------|-------------------|------|
| `pre_completed` | Agent B 评审失败 | **→ `waiting_user`** | ⭐ 转人工确认，不要草率标记为 completed |
| `pre_need_support` | Agent B 评审失败 | → `waiting_user` | 保持不变 |

  }
}

// 新增：标记为 waiting_user 的辅助方法
private async markTaskWaitingUser(
  task: typeof agentSubTasks.$inferSelect,
  reason: string
) {
  console.log(`[状态机] 标记为 waiting_user: ${task.id}, 原因: ${reason}`);
  
  await db
    .update(agentSubTasks)
    .set({
      status: 'waiting_user',
      metadata: {
        ...task.metadata,
        waitingReason: reason,
        failedAt: getCurrentBeijingTime(),
      },
      updatedAt: getCurrentBeijingTime(),
    })
    .where(eq(agentSubTasks.id, task.id));
}

### 场景 3：数据库操作失败（更新状态时出错）- 修正版

**逻辑：**
- 数据库操作失败 → 这个比较特殊
- 需要重试机制，或者记录日志
- 如果重试也失败，再转人工

**代码执行流程：**

```typescript
private async processPreCompletedReviewResult(
  task: typeof agentSubTasks.$inferSelect,
  reviewResult: AgentBReviewResult
) {
  switch (reviewResult.decision) {
    case 'NEED_REVISE':
      console.log(`[状态机] Agent B 要求修改，回到 pending 状态`);
      
      // 尝试更新数据库，带重试
      let retryCount = 0;
      const maxRetries = 3;
      
      while (retryCount < maxRetries) {
        try {
          await db
            .update(agentSubTasks)
            .set({
              status: 'pending',
              metadata: {
                ...task.metadata,
                retryCount: (task.metadata?.retryCount || 0) + 1,
                agentBRevision: reviewResult.data?.revisionInstructions,
              },
              updatedAt: getCurrentBeijingTime(),
            })
            .where(eq(agentSubTasks.id, task.id));
          
          // ✅ 成功
          return;
          
        } catch (dbError) {
          retryCount++;
          console.error(`[状态机] 数据库更新失败 (尝试 ${retryCount}/${maxRetries}):`, dbError);
          
          if (retryCount >= maxRetries) {
            // ❌ 重试也失败了，转人工
            throw new Error(`数据库更新失败，已重试 ${maxRetries} 次`);
          }
          
          // 等待 1 秒后重试
          await new Promise(resolve => setTimeout(resolve, 1000));
        }
      }
      break;
  }
}
```
